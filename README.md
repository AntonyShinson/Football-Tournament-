# FootballHub

https://football-tournament-drab.vercel.app/

A web-based football tournament management system that automates tournament organization from registration to the final match.

## Features

- No login or account required
- Team registration and approval
- Automatic group generation
- Round-robin fixtures
- Live standings and points table
- Score submission and confirmation
- Automatic knockout bracket
- Public tournament page
- Mobile-friendly interface
